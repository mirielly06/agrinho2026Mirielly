4. Infográfico Web Interativo: Do Lixo ao Biofertilizante
O Problema: A destinação incorreta de resíduos orgânicos da produção animal (como de suínos ou aves) causa severa poluição ambiental.
O que o código faz: Uma página única (Single Page) altamente visual que funciona como um infográfico animado sobre o funcionamento de um biodigestor nas propriedades rurais.
O Equilíbrio: O aluno usa a estrutura semântica do HTML5 (<section>, <article>, <figure>) aliada a animações em CSS (@keyframes) para mostrar graficamente o ciclo: o resíduo entra \(\rightarrow \) o gás vira energia \(\rightarrow \) a matéria orgânica vira biofertilizante. É uma excelente demonstração visual do conceito de economia circular no campo.
